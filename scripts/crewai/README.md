# CrewAI Scripts

This directory contains scripts and configuration files for CrewAI integration with the Beautiful Timetracker App.

## Files

- `crewai_config.py` - CrewAI configuration
- `crewai_example.py` - Example CrewAI usage
- `crewai_usage.py` - CrewAI usage utilities
- `activate_crewai.sh` - Script to activate CrewAI environment
- `verify_crewai_setup.py` - Script to verify CrewAI setup
- `requirements.txt` - Python dependencies for CrewAI
- `CREWAI_README.md` - CrewAI documentation
- `CREWAI_INTEGRATION_COMPLETE.md` - Integration completion notes
- `.crewai-review-checklist.md` - Review checklist

## Setup

1. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Activate the CrewAI environment:
   ```bash
   ./activate_crewai.sh
   ```

3. Verify setup:
   ```bash
   python verify_crewai_setup.py
   ```
